# Notion Operating Model

Use Notion as the human-readable playbook and Codex files as the executable toolkit.

- Start Here

## What Belongs In Notion

- Human-readable explanations.
- Diagrams, examples, and checklists.
- Decisions and rationale.
- Reusable patterns learned from projects.

## What Belongs In Codex Files

- `AGENTS.md` rules.
- `SKILL.md` workflows.
- Reference files Codex should load on demand.
- Scripts and custom agent templates.
